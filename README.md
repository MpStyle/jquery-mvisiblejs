# jQuery - MVisibleJS

## Using
```html
<script type="text/javascript" src="dist/jquery-mvisiblejs.min.js"></script>
```

## Development
- Clone the repository
- Install NPM
- Install dev dependencies
```
npm install typings --global
```
- In the root of the project run:
```
npm install
```
And
```
typings install --ambient
```
To build run:
```
grunt
```
